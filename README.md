# notify-gtk. DdShurick, GPL v2.
Ключи -i | --info, -e | --error 
Выводит текст на экран и удерживает заданное время (ключ -n | --timeout)
Ключи -q | --question,  -w | --warning
Выводит текст на экран с выбором "Да|Нет"
Ключ -s | --start запускает заданную программу и удерживает сообщение до окончания работы.

Компилировать: make
Установить: make install

Для gtk3 "sed -i 's/2/3/g' Makefile"
